Stylish Online Portfolio


Concise Summary:

Concept and Inspiration:

I was hired to make a website for a client who wanted to showcase her photography portfolio. This was a two-month project start to finish.

Team and Team Size:

This project was developed entirely by myself. I created the layout of webpages and added in whatever text/images my client asked to be on the site.

Main design goals:

  • Color pallete that looked visually appealing
  • Galleries of photos shown via a carousel
  • Responsive design that functions well on mobile and desktop
  • Easy layout that a technologically ignorant person can understand and navigate

    • Main Challenges Faced:

  • Creating the carousel feature
  • Maintaining similar image sizing on any given page

    • Tools and Technologies Used:

  • Visual Studio Code
  • Html5
  • CSS3
  • Javascript
