2D Plaformer Game "Hardcore Parkour!"


Concise Summary:

Concept and Inspiration:

This was my first game I've made. I wanted to learn the basics of game development as well as the Unity game engine.


Team and Team Size:

This was a solo project, as I had absolutely no experience in any aspect of making a game. Extra teammates would have added more complexity.

Main design goals:

  • Have a character that the player can control
  • Have parkour levels
  • Have coins that are collectable and add to your score

    • Main Challenges Faced so Far:

  • Learning the Unity Engine
  • Learning How to Animate a Character
  • Making a tutorial to explain movement

    • Tools and Technologies Used:

  • Unity Game Engine
  • C#
  • Unity Asset Store for Background Art
